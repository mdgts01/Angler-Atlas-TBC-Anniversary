# Angler Atlas – TBC Anniversary Edition

This repository contains a modified version of **Angler Atlas**, originally created by **deahgib**, updated to work with **World of Warcraft: The Burning Crusade Anniversary**.

The purpose of this fork is to preserve and extend the functionality of the addon for the Anniversary version of TBC, while maintaining the spirit and design of the original project.

## Changes

This version includes several updates and adjustments, including:

* Updated for **TBC Anniversary** compatibility
* Added support for newly introduced fish
* Added support for newly introduced fishing-related items
* Updated data to better reflect the current Anniversary environment
* Minor fixes and adjustments

Additional improvements may be added in future updates as development continues.

## Credits

All credits for the original addon, including its design, functionality, data structure, and original development, belong entirely to **deahgib** and any other original contributors.

This repository is intended solely as a compatibility and maintenance fork for the TBC Anniversary community.
